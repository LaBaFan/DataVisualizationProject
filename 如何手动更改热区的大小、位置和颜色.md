## 📐 怎么改热区的大小、位置和颜色

### 文件位置

```
src/data/overallHotspots.ts    ← 改大小和位置
src/styles/global.css          ← 改颜色
```

### 改大小和位置

打开 `src/data/overallHotspots.ts`，每个热区的 `coords` 数组控制位置和大小：

**圆形热区**（天气模块用的多）：
```typescript
{
  id: 'sunny_hotspot',
  type: 'circle',
  coords: [1450, 210, 74],    // [圆心X, 圆心Y, 半径]
  //         ↑      ↑    ↑
  //       横坐标  纵坐标  大小
}
```
- 第 1 个数字：左右位置（0=最左，1600=最右）
- 第 2 个数字：上下位置（0=最上，1000=最下）
- 第 3 个数字：半径（越大圈越大）

**矩形热区**（区域模块用的多）：
```typescript
{
  id: 'dispatch_center_hotspot',
  type: 'rect',
  coords: [660, 410, 260, 170],    // [左上角X, 左上角Y, 宽度, 高度]
  //         ↑      ↑    ↑     ↑
  //       横坐标  纵坐标  宽   高
}
```
- 第 1 个数字：左边距
- 第 2 个数字：上边距
- 第 3 个数字：宽度
- 第 4 个数字：高度

**多边形热区**（不规则形状）：
```typescript
{
  id: 'restaurant_street_hotspot',
  type: 'polygon',
  coords: [180, 520, 410, 470, 520, 590, 450, 755],    // [x1,y1, x2,y2, x3,y3, ...]
}
```
- 每两个数字是一个顶点的坐标，按顺序连起来

所有坐标都在 **1600 × 1000** 的 viewBox 里。你可以先把半径/宽高改大一点试试效果，调好再改回来。

### 改颜色

打开 `src/styles/global.css`，搜索 `.overall-hotspot`：

```css
/* 默认状态 */
.overall-hotspot {
  fill: rgb(16 185 129 / 0.18);     /* 填充色，改这里的 0.18 调透明度 */
  stroke: rgb(16 185 129 / 0.6);    /* 描边色 */
  stroke-width: 2.5;                /* 描边粗细 */
}

/* 鼠标移上去 */
.overall-hotspot:hover {
  fill: rgb(16 185 129 / 0.32);
  stroke: rgb(5 150 105 / 0.9);
  stroke-width: 3.5;
}
```

颜色对照表（常用的）：
| 颜色 | RGB 值 |
|------|--------|
| 绿色（当前） | `rgb(16 185 129)` |
| 蓝色 | `rgb(59 130 246)` |
| 紫色 | `rgb(168 85 247)` |
| 橙色 | `rgb(249 115 22)` |
| 红色 | `rgb(239 68 68)` |

改完后运行 `npm run build` 验证，然后 `npm run dev` 看效果。
